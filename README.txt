Amazon reviews recommender systems
Costruire sistemi di raccomandazione con approccio collaborative filtering item-based per dati provenienti dal sito Amazon.com.

Prerequisiti
Installare Python nella propria macchina. Andare su https://www.python.org/downloads e cliccare su una delle ultime versioni disponibili. 
Una guida per l’installazione e la configurazione è disponibile al seguente link: https://realpython.com/installing-python/. 
Scaricare le librerie utilizzate per le analisi: pandas, numpy, scipy, gzip, json, sys, random, time.
Se non si dispone di esse è possibile scaricarle utilizzando pip e la seguente istruzione da linea di comando pip install <nome_libreria>
Se pip non è installato nella propria macchina si suggerisce di seguire questo tutorial: https://github.com/BurntSushi/nfldb/wiki/Python-&-pip-Windows-installation#pip-instal.
Accertarsi di possedere nella macchina una delle seguenti applicazioni: se si lavora su Windows: Windows PowerShell o Prompt dei comandi, se si lavora su Mac: Terminale.

Eseguire il codice
Portarsi, da linea di comando, all'interno della cartella dove è contenuto il file finale.py ed eseguirlo utilizzando il comando python finale.py. 
I risultati verrano stampati di seguito.
Il programma esegue un'analisi sui dati contenuti nel file Digital_Music.csv. 
Qualora si desideri lavorare su altri dati, scaricabili da https://nijianmo.github.io/amazon/index.html e scaricare il file d'interesse di tipo 5-core. 
Indicare all'interno di finale.py il nome di questo. 
Il file che si indica deve essere di tipo .csv. Utilizzare Apertura.py per convertirlo in .csv. Tale operazione è stata svolta anche per Digital_Music.csv
